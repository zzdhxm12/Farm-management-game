# SimFarm
 Farm management game
 
 동물농장을 구성, 각종 동물들을 잘 키워 직접 판매하거나 부산물(계란, 우유 등)을 판매하고 능력치를 향상시켜 경진대회에서 우승하는 기능을 포함한 프로그램
